Working with HTML5 Input Types

Input type="email" accepts email address

Input type="text" names username etc

Input type="color" accepts color

Input type="radio" to select one from two or more options

Input type="checkbox" one or more options

Input type="number" accepts only numeric values

Input type="url" accepts website address

Input type="range" to specify a range

Input type="tel" accepts telephone numbers

input type="date" accepts dates
